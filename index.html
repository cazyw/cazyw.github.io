File Comparison/Backup (Python)

This was a personal project. I needed a simple backup program I could use to backup certain folders. All the programs on the market were quite complicated, all I wanted was something that would:

  1. give me a listing of the files in a folder
  2. compare the files in two folders and show me which were new/updated and would be copied over
  3. copy only the new/updated files

I built this program from scratch, initially so that it would run on the command line and later adding a GUI. Particularly challenging in this project was resarching how to access system files and timestamps and how to build a GUI interface. It was initially written in Python 2 but has since been upgraded to Python 3 and is intended to be run in a Windows environment.

Webpage Creation from Text Files (Python)

This was a personal project. I was building a website where I was reviewing episodes of Doctor Who. But every time I changed the style or layout of the page and added different features, I had to change the code for every page (which was very time consuming and frustrating).

To cut down on repetition, I wrote a script to dynamically create the html files. I wrote my review for each episode in separate text files, organised in a specific file structure and built a python script that would build the webpages based on the text files. This way I only needed to amend this script to make changes across all pages.

The challenging aspect of this project was looping through and traversing folder structures and using regular expressions to dynamically match patterns to create the different HTML pages.

Microblog App (Ruby on Rails)

Building a Ruby on Rails App with this tutorial: Ruby on Rails Tutorial: Learn Web Development with Rails by Michael Hartl. This was my introduction to the Model View Controller (MVC) pattern and using Rails. Following this tutorial took me through the steps of building a twitter-like app which allowed users to: create an account (including the use of secure passwords), receive an activation email (using SendGrid), log in, post a micro blog including pictures, follow and unfollow other users, and for users with admin access, delete other users.

This tutorial also introduced me to writing automated tests, the test driven development, and the "Red, Green, Refactor" cycle.

The App is currently deployed and available on Heroku.
